# CurveShadowCss
you can use class="curve-shadow-effect" to use this curve shadow effect
#example
    <!DOCTYPE>
    <html>
    	<head>
    		<title>css实现曲线阴影</title>
    		<link style="text/css" rel="stylesheet" href="./curveshadow.css">
    	</head>
    
    	<body>
    	<div class="wrap curve-shadow-effect">
    		<h1>shadow effect</h1>
    		</div>
    	</body>
    </html>
